# プロダクトゴール

## ビジョン

AWS サーバーレスアプリケーションを運用する個人開発者・小規模チームが、アプリケーションの健康状態を**一目で把握**し、問題を**迅速に特定・解決**できる、シンプルで低コストなオブザーバビリティダッシュボードを提供する。

## プロダクトゴール

> **サーバーレスアプリの運用状態を、月額 $1 以下のコストで、誰でも簡単に可視化できるオブザーバビリティプラットフォーム「Serverless Insight」を構築する。**

## 成功指標

| 指標 | 目標値 |
|------|--------|
| ダッシュボード初期表示時間 | 3 秒以内 |
| API レスポンス時間 | 5 秒以内（CloudWatch API 呼び出し含む） |
| 月額運用コスト | $1 以下 |
| 対応画面数 | 4 画面（Overview / Endpoints / Endpoint Detail / Logs） |

## 対象ユーザー

- **主ターゲット:** AWS でサーバーレスアプリを運用する個人開発者
- **セカンダリ:** 1〜5 名規模の小規模チーム（商用 APM 導入前段階）

## 提供価値

1. **簡単さ** — CloudWatch の複雑な UI を介さずに、必要な情報に素早くアクセスできる
2. **低コスト** — AWS サービスのみで構成し、月額 $1 以下で運用可能
3. **即座の把握** — アプリ全体の健康状態、API 単位のパフォーマンス、エラーログを一目で確認できる

## スコープ

### MVP に含めるもの

- 監視対象となる Demo Web App（投稿 CRUD アプリ）
- Observability Platform（4 画面構成のダッシュボード）
- CloudWatch メトリクス・ログの取得と可視化
- 全インフラの Terraform 管理

### MVP に含めないもの

- 分散トレーシング（X-Ray 連携）
- 認証・認可
- 自動リロード / リアルタイム更新
- マルチテナント / マルチリージョン対応
- 高度なアラート・通知機能
